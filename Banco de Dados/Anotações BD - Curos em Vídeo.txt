---- A origem dos banco de dados ----
-> década de 50 os dados eram guardados em papel, as fichas são registros, pastas tabelas, armarios arquivos 
-> Os dados eram guardados em arquivos sequencial
-> Após surgiu os discos e hds. 
-> década de 60 o Departamento de Defesa dos EUA, 
criou uma tarefa para armazenar dados de maneira segura e eficiente
-> Criou um evento CODASIL
-> Surgimento da linguagem COBOL -> lógica e estruras de dados.
-> Surgimento de uma nova tecologia -> Banco de dados
-> BD é composto pela Base da Dados, Sistema Gerenciador, Linguagem de Exploração, Programas Adicionais
-> IBM valiosa para construção do Banco de Dados
-> Dados hierárquicos de maneira simplista -> primeiro modelo proposto pela IBM
-> Modelos em rede -> Segundo pela IBM
-> Década de 70 propos um novo modelo com relacionamentos
-> Modelo Relacional
-> SEQUEL
-> SQL - (Structured Query Language) linguagem de consulta
-> Oracle, IBM, dBase, SQL Server
-> MySQL - gratuito que pertence a Oracle
-> MariaDB, PostGressSQL
